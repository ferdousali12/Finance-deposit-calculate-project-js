# Day 11: FD Maturity Calculator 💰

A simple, clean, and functional Maturity Amount Calculator built with HTML, CSS, and JavaScript. This tool helps users calculate the total returns on their investments based on principal, interest rate, and tenure.

## 🚀 Live Demo
[View Project Live](https://soft-arithmetic-9f689f.netlify.app/)

## 🛠️ How it Works
The app uses the Simple Interest formula to determine the final amount:
$$A = P \left(1 + \frac{rt}{100}\right)$$

Where:
- **P** = Principal Amount
- **r** = Annual Interest Rate
- **t** = Tenure (Years)

- **JavaScript**: Calculation Logic & DOM Manipulation
